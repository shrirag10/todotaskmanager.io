# Todo Task Manager

A minimal frontend todo application. Add, complete, and remove tasks — no backend required.

**Live**: [shrirag10.github.io/todotaskmanager.io](https://shrirag10.github.io/todotaskmanager.io/)

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- Deployed via GitHub Pages

## Run Locally

Open `index.html` in a browser — no build step needed.
